pub fn lambert_w0(z: f64) -> f64

The principal branch of the Lambert W function, W_0(z).

Returns NAN if the given input is smaller than -1/e (≈ -0.36787944117144233).

Accurate to 50 bits.

Wrapper of lambert_w_0 function of lambert_w crate.
